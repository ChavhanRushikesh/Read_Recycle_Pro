# Read Recycle Project

A web-based application built using **PHP, HTML, CSS, JavaScript, and MySQL**. The project runs on the **XAMPP** local server and uses **MySQL** for database management.

## Features

* User-friendly interface
* PHP backend for server-side processing
* MySQL database integration
* Responsive design using HTML and CSS
* Interactive functionality with JavaScript

## Technologies Used

* PHP
* HTML5
* CSS3
* JavaScript
* MySQL
* XAMPP (Apache + MySQL)

## Prerequisites

Before running the project, make sure you have:

* XAMPP installed
* PHP 8.x or later (or the version used for development)
* MySQL
* A web browser

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git
   ```

2. Move the project folder to the **htdocs** directory inside your XAMPP installation.

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.

4. Create a new database in phpMyAdmin.

5. Import the provided SQL file (if included) into the database.

6. Update the database configuration in your PHP files if necessary.

7. Open your browser and visit:

   ```
   http://localhost/your-project-folder/
   ```
